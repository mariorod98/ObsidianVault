# Práctica 2 TCVD

## Requisitos dataset
Tienen que contener variedad de datos numéricos y categóricos, actualmente no tenemos suficientes datos numéricos. 

Propongo obtener el siguiente dataset:
- Nombre
- Tipo (sobremesa, portatil) : cualitativo
- Marca : cualitativo
- Número de cores CPU : numérico entero
- Frecuencia CPU (MHz): numérico real
- Número de cores GPU: numérico entero
- Frecuencia GPU (MHz): numérico real 
- Capacidad RAM (GB) : numérico entero
- Capacidad disco duro (GB) : numérico entero
- Resolución pantalla ancho (pixel): numérico real
- Resolución pantalla alto (pixel) : numérico real
- Tamaño pantalla (pulgadas) : numérico real
- Precio  (euros): numérico real

**Problema**: PCComponentes no muestra algunos de estos datos, especialmente sobre GPU y CPU.

**Alternativa**: O bien hacemos scrapping de otras paginas webs o encontramos un dataset que contenga estos datos.
[TechPowerUp](https://www.techpowerup.com/) contiene una base de datos de componentes de PC. En principio no he visto que restrinja su uso para uso académico, pero podemos preguntar por una licencia de uso académico para acceder a sus datos.

Esto además nos dará muchos más datos con los que trabajar.
