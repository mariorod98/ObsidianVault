# Nivel de Portal
**Objetivo:** Abrir la puerta activando dos botones a presión.
**Mecánicas principales.**
- Botones de presión: Serán utilizados para activar las puertas usando Companion cubes.
- Rayo desintegrador: Serán utilizados como obstáculos para impedir que el cubo llegue al botón de presión.
- Companion cube: Será utilizado para activar los botones a presión.

Puzzle para activar el primer botón:
El primer botón se encuentra tras una pantalla
