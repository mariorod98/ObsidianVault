# UE4 Naming Prefixes
- Classes derived from AActor are prefixed with A.
- Classes derived from UObject are prefixed with U (e.g., UComponent).
- Enums are prefixed with E.
- Interface classes are prefixed with I.
- Template classes are prefixed with T.
- Classes derived from SWidget are prefixed with S.
- Everything else is prefixed with F.
