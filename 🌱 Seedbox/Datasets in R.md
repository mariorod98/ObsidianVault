# Datasets in R
- `head`: To see the first values.
- `tail`: To see the last values.
- `str`: To see the structure.
- `summary`: To see basic statistics of each column

## Acces dataset info
To access info use `[rows,columns]`. You can leave one of the two parts empty to selects all the values of that part
