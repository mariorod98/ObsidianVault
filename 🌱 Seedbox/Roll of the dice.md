# Roll of the dice
## Pitch
Un battle arena. Tienes un grid de juego en el que esta tu personaje y varios enemigos (posiblemente elementos del entorno tambien).

Se trata de un juego por turnos. Primero el turno del jugador. Luego el de los enemigos.

Al inicio de su turno, el jugador lanza una serie de dados (d6). Con estos dados, puede realizar las distintas acciones.

Los enemigos tienen acciones simples. Distintos tipos de enemigos realizan distintos tipos de acciones. No utilizan dados.

Puede haber distintas clases de personaje. Cada personaje deberá tener una ventaja y una desventaja.

## Acciones
Las acciones que puede realizar el jugador son:
- **Moverse**. el jugador puede gastar un dado para moverse tantas casillas como el dado indica. Puede moverse menos casillas si así lo desea.
- **Atacar**. El jugador puede atacar a un enemigo que esté en el rango de ataque. Dos formas distintas de realizar el ataque:
	- El dado puede ser una probabilidad de que el ataque se realice con éxito. Cuanto mayor sea el dado, mayor la probabilidad. El daño que realiza al enemigo es siempre el mismo. Se puede incluir probabilidad de critico. Ventaja: introduce mecánica riesgo/recompensa ya que el ataque puede fallar, acertar o ser crítico. El ataque puede depender del arma que lleve el personaje, dando más variabilidad al combate. Desventaja: el jugador puede verlo como menos justo.
	- El dado es el daño plano que realiza al enemigo. No hay probabilidades. Ventaja: el jugador sabe a ciencia cierta cuanto daño va a realizar, no hay riesgo. Desventajas, el juego tiene menos variabilidad y se elimina el riesgo/recompensa de atacar.
- **Protegerse**. El jugador puede gastar dados para añadir puntos de vida temporales. 

## Personajes
### Personajes jugables
- **Guerrero**.
	- Ventaja. Una vez por turno, puede repetir una tirada de ataque fallida.
	- Desventaja. ????
- **Arquera**. 
	- Ventaja. Si el enemigo está a x de distancia, recibe más daño o tiene más probabilidad de acertar. 
	- Desventaja. Si está a menos distancia, recibe menos daño o tiene menos probabilidad de acertar.
- **Pícaro**. 
	- Ventaja. Sus ataques le curan y puede moverse al doble del valor del dado. 
	- Desventaja. No se puede proteger.

### Enemigos
En principio, los enemigos no tienen dados. Realizan acciones simples. Cada enemigo puede realizar estas dos acciones en su turno:
- Desplazarse. Cada enemigo puede desplazarse un numero máximo de casillas. Este número depende del enemigo.
- Atacar. El enemigo puede atacar una vez por turno. El ataque, en principio, siempre impacta.
- **Masilla**. Un enemigo que ataca cuerpo a cuerpo y se desplaza lentamente. Individualmente no supone ningún peligro, pero en grupo pueden resultar letales.
- **Masilla a distancia**. Como el masilla, pero ataca a distancia. Tiene mucha menos vida.
- **Clérigo**. Unidad que sirve de apoyo al resto. Una vez por turno, puede curar a un aliado.
- **Tanque**. Tiene mucha más vida pero un daño muy bajo. Puede ponerse armadura.
- **Kamikaze**. Tiene poca vida. Si llega a estar cuerpo a cuerpo con el jugador, se inmola provocando mucho daño en un area. Si muere, también provoca ese daño.
