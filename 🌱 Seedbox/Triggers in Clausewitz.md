# Triggers in Clausewitz
Triggers are a game logic that checks conditions about the game state.

Triggers may be used to determine when an [[Effects in Clausewitz|effect]] or a data structure should be executed or crated. In these situations, they implicitly return *yes* or *no*. 

## Trigger modifiers
- **NOT**
- **AND**
- **OR**
- **NAND**
- **NOR**
