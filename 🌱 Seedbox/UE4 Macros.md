# UE4 Macros
- **UCLASS( )**: Used to tell Unreal to generate reflection data for a class. The class must derive from [[UE4 Core Classes#UObject|UObject]].
- **USTRUCT( )**: Used to tell Unreal to generate reflection data for a [[UE4 Core Classes#UStruct|UStruct]].
- **GENERATED_BODY( )**: Replaced in UE4 with all the necessary boilerplate code that gest generated for the type.
- **UPROPERTY ()**: Enables a member variable of a UCLASS or a USTRUCT to be used as a [[UPROPERTY]]-
- **UFUNCTION( )**: Enables a class method of a UCLASS or a USTRUCT to be used as a [[UFUNCTION]].
- **UENUM( )**: Used to tell Unreal to generate reflection data for an enum.
