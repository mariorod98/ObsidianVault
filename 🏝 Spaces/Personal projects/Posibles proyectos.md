### Roll of the dice
Un battle arena. Tienes un grid de juego en el que esta tu personaje y varios enemigos (posiblemente elementos del entorno tambien).

Se trata de un juego por turnos. Primero el turno del jugador. Luego el de los enemigos.

Al inicio de su turno, el jugador lanza una serie de dados (d6). Con estos dados, puede realizar las distintas acciones.

Las acciones que puede realizar el jugador son:
- Moverse. el jugador puede gastar un dado para moverse tantas casillas como el dado indica. Puede moverse menos casillas si así lo desea.
- Ataque principal. El jugador puede atacar a un enemigo que esté en el rango de ataque. Dos formas distintas de realizar el ataque:
	- El dado puede ser una probabilidad de que el ataque se realice con éxito. Cuanto mayor sea el dado, mayor la probabilidad. El daño que realiza al enemigo es siempre el mismo. Se puede incluir probabilidad de critico. Ventaja: introduce mecánica riesgo/recompensa ya que el ataque puede fallar, acertar o ser crítico. El ataque puede depender del arma que lleve el personaje, dando más variabilidad al combate. Desventaja: el jugador puede verlo como menos justo.
	- El dado es el daño plano que realiza al enemigo. No hay probabilidades. Ventaja: el jugador sabe a ciencia cierta cuanto daño va a realizar, no hay riesgo. Desventajas, el juego tiene menos variabilidad y se elimina el riesgo/recompensa de atacar.
- Defensa. El jugador puede gastar dados para añadir puntos de vida temporales. 

Los enemigos tienen acciones simples. Distintos tipos de enemigos realizan distintos tipos de acciones.
