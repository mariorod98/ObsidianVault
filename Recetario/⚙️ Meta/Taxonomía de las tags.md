# Taxonomía de las tags

Lista de tags que se usa en [[Recetario]].

## Duracion
- `#duracion/rapido` para recetas de menos de 15 minutos.
- `#duracion/medio` para recetas de entre 15 y 40 minutos.
- `#duracion/largo` para recetas de entre 40 y 90 minutos.
- `#duracion/muyLargo` para recetas de más de 90 minutos.

## Dificultad
- `#dificultad/facil` para recetas que no tienen elaboraciones complejas.
- `#dificultad/medio` para recetas que requiere alguna elaboracion compleja.
- `#dificultad/dificil` para recetas con elaboraciones muy complejas.

## Hora del día
- `#hora/desayuno` para recetas que prefiero tomar en el desayuno.
- `#hora/almuerzo` para recetas que prefiero tomar en el almuerzo.
- `#hora/cena` para recetas que prefiero tomar en la cena.

## Tipo de plato
- `#tipo/cuchara` para recetas que son de cuchara.
- `#tipo/cuchara/sopa` para sopas.
- `#tipo/cuchara/guiso` para guisos y cocidos. 
- `#tipo/cuchara/pure` para purés.
- `#tipo/dip` para dippeos.
- `#tipo/salsa` para salsas.
- `#tipo/reposteria` para reposteria.
- `#tipo/reposteria/bizcocho` para bizcochos.
- `#tipo/ensalada` para ensaladas.
- `#tipo/masa` para masas.

## Elaboraciones
- `#elaboraciones/horno`
- `#elaboraciones/olla`
- `#elaboraciones/ollaPresion`
- `#elaboraciones/thermomix`
